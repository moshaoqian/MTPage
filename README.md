# MTPage
模仿豆瓣评论弹出页面

## 介绍
MTPage是我之前项目的代码，最近开源出来供大家使用，当时是模仿豆瓣的评论页跳转做的，它是一种多评论跳转的解决方案，不论评论数量多少，它都能进行处理。
